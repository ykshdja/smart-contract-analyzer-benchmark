#TRANSAACTION LIFECYCLE 
- user calls - withdraw(1 ETH)
- user clicks  withdraw in wallet (MetaMask)
- user signs Transaction (with private key)
-- this proves ownership
- Transaction sent to ETH Network
-Validators verify
- If Verified -> Transaction added to block
- If Transaction calls a contract - Every validator executes the code.
- Block chain is updated
- Finalized Transaction becomes Immutable
